# Healthcare Readmission Prediction System

## Project Overview
This project develops an end-to-end healthcare analytics and machine learning pipeline to predict 30-day hospital readmission risk using the Diabetes 130-US Hospitals dataset.

The workflow includes:
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Feature engineering
- Predictive modelling
- Risk stratification
- Dashboard visualisation
- Business impact analysis

---

## Dataset
- Source: UCI Machine Learning Repository
- Dataset: Diabetes 130-US Hospitals Dataset
- Records: 100,000+ patient records

---

## Technologies Used
- Python
- pandas
- NumPy
- scikit-learn
- matplotlib
- seaborn
- Jupyter Notebook

---

## Machine Learning Models
- Logistic Regression
- Random Forest Classifier

---

## Key Features
- 30-day readmission prediction
- ROC curve analysis
- Confusion matrix evaluation
- Feature importance analysis
- Patient risk stratification
- Executive healthcare dashboard

---

## Key Insights
- Previous inpatient admissions strongly influenced readmission risk.
- Random Forest achieved better predictive performance compared to Logistic Regression.
- Risk stratification effectively identified high-risk patient groups.
